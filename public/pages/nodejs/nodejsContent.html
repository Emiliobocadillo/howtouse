Node.js

What is Node.js?

Node.js is a runtime environment to run javascript outside of the browser - which means it can run on a server. This is how it works: a GET request is made, by someone visiting a URL that points to the node.js server, when the request is recieved node.js can handle the request and read the appropriate file from the server's file system, and then respond back to the client so they can view the requested site in their browser.

Get started

Navigate to https://nodejs.org/en and download the LTS version (long term support version). Complete the installation wizard to install Node.js and npm (Node Package Manager).


To check if the installation completed succesfully write the following command in your terminal:


Terminal
          
node --version
If the terminal outputs a version, the installation went succesfully.
